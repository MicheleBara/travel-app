ℹ️ Come trasferire i dati sull'iPad

  1. Clicca Scarica backup su questo PC
  2. Trasferisci il file sull'iPad (via AirDrop, email, o iCloud Drive)
  3. Apri l'app sull'iPad e vai su Backup → Importa
  4. Seleziona il file scaricato
